What is web design?


Web design is a subject that involves the development of websites. Sites like Youtube, Facebook, and even Google Classroom run on 3 languages:


  1. HTML: The language that displays text, colors, and images.
  2. CSS: The language that styles the HTML data, making the website more visually appealing.
  3. JavaScript: The backend language that focuses on the website's functions.



Key Concepts:


  1. User Interface (UI) Design: Focuses on creating visually appealing layouts that are easy for users to interact with.
  2. Accessibility: Ensures the website can be used by everyone, including people with disabilities, through proper structure and tags.
  3. Web Usability: Keeps the design simple and clear so visitors can easily find what they need and understand the content.
  4. Information Architecture: Organizes website content and pages logically to make navigation easy for both users and search engines.
  5. Web Security: Protects user data and the site from attacks using HTTPS, input validation, and regular updates.
